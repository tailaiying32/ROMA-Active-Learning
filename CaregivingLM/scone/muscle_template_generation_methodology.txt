MUSCLE PARAMETER TEMPLATE GENERATION METHODOLOGY
===============================================

OVERVIEW
--------
This document describes the systematic approach for generating 250 muscle parameter templates 
to achieve comprehensive coverage of clinically plausible muscle weakness patterns for 
reaching behavior simulation.

OBJECTIVES
----------
1. Generate muscle parameter templates with SIGNIFICANT weaknesses that impact function
2. Ensure COMPLETE COVERAGE of clinically relevant muscle weakness combinations
3. Maximize functional diversity in reaching behavior outcomes
4. Maintain clinical plausibility of all generated patterns

MUSCLE ORGANIZATION
------------------
The 40 muscles are organized into 8 functional groups based on anatomical function:

1. SCAPULAR STABILIZERS (10 muscles):
   - trap_scap_m_r, trap_scap_s_r, trap_scap_i_r, trap_clav_s_r
   - ser_ant_i_r, ser_ant_m_r, ser_ant_s_r
   - rhom_s_r, rhom_i_r, lev_scap_r

2. ROTATOR CUFF (5 muscles):
   - supraspi_r, infraspi_i_r, infraspi_s_r, teres_min_r
   - subscap_s_r, subscap_i_r

3. DELTOID COMPLEX (3 muscles):
   - delt_clav_a_r, delt_scap_p_r, delt_scap_m_r

4. LARGE SHOULDER MOVERS (6 muscles):
   - pect_maj_clav_s_r, pect_maj_thorax_i_r, pect_maj_thorax_m_r
   - lat_dors_m_r, lat_dors_i_r, teres_maj_r

5. SMALL SHOULDER MUSCLES (2 muscles):
   - pect_min_r, coracobrach_r

6. ELBOW MUSCLES (5 muscles):
   - bic_long_r, bic_brev_r, brach_r
   - tri_long_r, tri_lat_r

7. FOREARM MUSCLES (3 muscles):
   - brachiorad_r, supinator_r, pronator_teres_r

8. WRIST MUSCLES (5 muscles):
   - ext_carpi_rad_long_r, ext_carpi_rad_brev_r, ext_carpi_ulna_r
   - flex_carpi_rad_r, flex_carpi_ulna_r

SEVERITY LEVELS
--------------
To ensure functional impact, only significant weakness levels are used:

- SEVERE (S): 0.1-0.4 (definite functional impact)
- MODERATE (M): 0.4-0.7 (likely functional impact)  
- NORMAL (N): 0.8-1.0 (preserved function, compensation possible)

NOTE: Mild weaknesses (0.7-0.8) are excluded as they may not produce 
      detectable functional differences due to compensation.

GENERATION STRATEGY: 4-TIER SYSTEMATIC COVERAGE
----------------------------------------------

TIER 1: SINGLE GROUP PRIMARY WEAKNESS (80 templates)
Each of 8 muscle groups × 10 templates = 80 total

Per muscle group distribution:
- 4 templates: Severe weakness (0.1-0.4)
- 3 templates: Moderate weakness (0.4-0.7)
- 3 templates: Combination patterns (some severe, some moderate within group)

Examples:
- Template 001: Scapular group severe (all 0.1-0.4), others normal (0.8-1.0)
- Template 011: Rotator cuff moderate (all 0.4-0.7), others normal
- Template 021: Deltoid mixed (anterior severe 0.2, posterior moderate 0.5, middle normal 0.9)

TIER 2: PAIRWISE COMBINATIONS (96 templates)
All muscle group pairs: C(8,2) = 28 pairs × 3-4 templates each

Severity combination patterns per pair:
- Both severe (S,S): 0.1-0.4 range
- One severe, one moderate (S,M): mixed severity
- One severe, others normal (S,N): isolated severe + normal
- Both moderate (M,M): 0.4-0.7 range

Examples:
- Template 081: Scapular severe + Rotator cuff severe
- Template 082: Scapular severe + Rotator cuff moderate  
- Template 083: Scapular severe + Rotator cuff normal (for comparison)
- Template 084: Scapular moderate + Rotator cuff moderate

TIER 3: CLINICAL SYNDROME PATTERNS (50 templates)
Based on real clinical conditions with known muscle involvement patterns.

STROKE SYNERGIES (15 templates):
- Flexor synergy patterns: Biceps strong, triceps weak; wrist flexors strong, extensors weak
- Extensor synergy patterns: Triceps strong, biceps weak; extensors strong, flexors weak
- Mixed patterns: Various combinations with different severities
- Severity variants: Mild, moderate, severe versions of each pattern

SCI PATTERNS (15 templates):
- C4 level: All arm muscles affected
- C5 level: Shoulder preserved, elbow/wrist affected  
- C6 level: Shoulder + biceps preserved, triceps/wrist affected
- C7 level: Above + triceps preserved, fine motor affected
- C8 level: Primarily hand/wrist affected
- Incomplete injury variants

NEUROLOGICAL CONDITIONS (10 templates):
- Cerebral palsy: Spastic patterns, hypotonia patterns
- Multiple sclerosis: Proximal weakness patterns
- Parkinson's disease: Rigidity patterns
- Brachial plexus: Upper trunk (C5-C6), lower trunk (C8-T1) patterns

ORTHOPEDIC CONDITIONS (10 templates):
- Rotator cuff tears: Isolated, massive, progressive
- Frozen shoulder: Capsular restriction patterns
- Impingement: Specific muscle inhibition patterns
- Nerve injuries: Axillary, radial, musculocutaneous patterns

TIER 4: COMPLEX MULTI-GROUP PATTERNS (24 templates)
Three or more muscle groups involved simultaneously.

THREE-GROUP COMBINATIONS (12 templates):
Most functionally critical triplets based on biomechanical analysis:
- Scapular + Rotator Cuff + Deltoid (shoulder complex)
- Deltoid + Elbow + Forearm (reaching chain)
- Large Movers + Small Shoulder + Elbow (power + precision)
- Other high-impact combinations

GLOBAL PATTERNS (6 templates):
- Proximal weakness: Scapular + Rotator Cuff + Deltoid weak, distal preserved
- Distal weakness: Elbow + Forearm + Wrist weak, proximal preserved  
- Mixed patterns: Alternating weak/strong groups

EXTREME PATTERNS (6 templates):
- Rare but functionally distinct combinations
- Single muscle isolations (one muscle severely weak, all others normal)
- Unusual compensation patterns

CLINICAL VALIDATION RULES
-------------------------
All generated templates must pass clinical plausibility checks:

1. ANATOMICAL CONSTRAINTS:
   - Synergistic muscles should have correlated weakness patterns
   - Antagonist muscles can have opposing patterns (weak/strong)
   - Multi-joint muscles affect multiple movement planes

2. NEUROLOGICAL CONSTRAINTS:
   - Nerve-specific patterns must follow anatomical innervation
   - Spinal level injuries follow dermatomal/myotomal patterns
   - Central patterns follow known synergy organizations

3. BIOMECHANICAL CONSTRAINTS:
   - Compensation patterns must be mechanically feasible
   - Force couples must maintain some balance for stability
   - Joint stability requirements must be considered

4. FUNCTIONAL CONSTRAINTS:
   - Patterns must produce distinguishable functional outcomes
   - Workspace limitations must be realistic
   - Movement quality changes must be plausible

PARAMETER ASSIGNMENT ALGORITHM
-----------------------------
For each template:

1. DETERMINE PRIMARY WEAKNESS PATTERN:
   - Identify target muscle groups from tier assignment
   - Assign severity levels based on template type

2. APPLY CLINICAL CONSTRAINTS:
   - Check synergy relationships within groups
   - Apply antagonist relationships between groups
   - Verify neurological plausibility if applicable

3. ASSIGN SPECIFIC VALUES:
   - Severe: Random uniform from [0.1, 0.4]
   - Moderate: Random uniform from [0.4, 0.7]  
   - Normal: Random uniform from [0.8, 1.0]
   - Apply small random variations to avoid identical values

4. VALIDATE TEMPLATE:
   - Check against clinical constraint rules
   - Verify functional distinctiveness
   - Ensure no mathematical impossibilities
